# 🚀 AI Resume Analyzer

AI-powered web app to analyze resumes and give feedback.

## 🔥 Features

* Upload resume (PDF/Image)
* Role-based analysis
* Strengths, Weaknesses, ATS score

## 🛠️ Tech Stack

Python, Streamlit, Gemini API
